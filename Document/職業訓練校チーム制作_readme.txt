要件定義
■システムの概要
本システムではJavaのservlet/JSPを用いて、
データベースと連動させつつ、
勤怠（出勤、退勤、休み等）を管理する
また、管理者はユーザー管理及び勤怠表を編集できる

■システムの目的
servlet/JSPを用いて、勤怠管理システムを作成する
データベースと連動させつつ、
エクセルによる入力及び出力に近づけることを目標とした

■開発メンバー・期間
作者含め5名
22日間

■機能
【一般ユーザー機能】
・ログイン
・ログアウト
・出退勤時間記録
・記録閲覧
・申請

【管理者のみ機能】
・ユーザー管理、勤怠管理の表示と編集
・新規ユーザー登録
・ユーザー情報の編集
・出勤時間、退勤時間、備考の編集
・備考欄にて一般機能からの申請を確認及び編集

■注意事項
画面左上のALL Whiteをクリックにより、メイン画面に遷移します
ログアウト後は、ログイン画面に遷移します
Eclipse 2021-06で実行する場合、
\pleiades\tomcat\9\libフォルダ内に
mysql-connector-java-8.0.26.jar
の追加が必要です

データベースのデータについては
create table.txt
を使用してください
最終行付近のselect文によりDBと連動していることが確認できます
データベースの削除については
drop table.txt
を使用してください

■実行環境
MySQL 8.0 Command Line Client
Eclipse 2021-06
Tomcat 9.0
Google Chrome
